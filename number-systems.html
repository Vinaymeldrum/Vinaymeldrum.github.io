Number Systems

Here, we will be discussing about the different types of number systems and how they relate to RAM.

Finite Prime Field

In a finite prime field, number of elements must be a prime. F_5 represents the number of elements that you are allowed. F_5 would be 5 elements, which would be 0, 1, 2, 3, and 4. It is similar to binary, however, instead of 0s and 1s, you can specify the element based on what F_(number) you have. The mathematical operation to express the finite field of F_5 is: modulus 5 (0, 1, 2, 3, 4)

4
+ 4
13

4+4 = 8. Mod 5 of 8 (the remainder of 5/8) is 3. Therefore, in the first column, you put 3, and in the second column, you put 1 because you have to carry it over.

Although related to computers, finite prime fields do not relate to RAM.

Binary

Computers communicate everything in 1s and 0s. The way this works is that you have to have a multiplier. Since there are 2 symbols, being 1 and 0, it means that for each column, you multiply by 2. An example is down below:

x      x      x      x     
8 4 2 1

If you wanted to represent the number 3 in binary, you would write it as 11. Because the first two spots are 2 and 1, by putting 1, it means that you want to add that place. If you wanted to represent the number 2, you would put: 10. In the 2nd column, the number 2 is represented. You do not need anything inside the 2nd column (1), so you put a 0. Binary is related to RAM because RAM is composed by a bunch of tiny bits. The RAM capacitor charges and discharges, storing either a 1 or 0 in the cell. However, the charge in this capacitor will slowly leak away, and must be refreshed periodically. Each byte communicates with the computer, holding temporary storage. This temporary storage is stored in binary, because it is the fastest as it is base 2. In RAM, the higher amount of bits, the more things you can temporary store. The data that you store can be retrieved in any order, hence the name Random Access memory. As soon as you turn off your computer you lose all your RAM storage. A more permanent way to store things in a computer is by using an SSD. An SSD is an average of around 1000 times slower than RAM storage.

ASCII

ASCII (American Standard Code for Information Interchange) is a character encoding standard for electronic devices. It represents text in computers etc. It was originally developed from telegraph code. Work on it began in October of 1960. The first standard was published in 1963. Unicode

Unicode is a standard way of representing characters on a computer. Common ways of encoding character are UTF-8, UTF-16 and UTF-32. It is a way to converting a sequence of characters to elements. It represents a set of characters, including ASCII characters in a coding sequence.


Although related to computers, Unicode does not relate to RAM.
