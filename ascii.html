ASCII

ASCII (American Standard Code for Information Interchange) is a character encoding standard for electronic devices. Because computers can only read numbers, ASCII was developed as a numerical representation of text. Originally derived from telegraph code, work on it began in the October of 1960. The first standard was published in 1963. It was revamped in 1967 and then updated once again in 1986.
