The Functions of RAM

RAM, or random access memory, is vital to a computer's functions.

RAM is used in computers as a place to store temporary data from the OS and other programs. It is more straightforward, accessing stored data directly as opposed to going through other data to get to what it needs. RAM is located near the procesor, making it extremely fast when retrieving data. It stores data by putting an electronic charge in a storage cell, which is replaced by a new charge every few milliseconds.

The RAM connects to the memory slot of the motherboard, but must conform to the proper slot type (DDR3, DDR4)
